## options:

* `--lat=[float]` latitude
* `--long=[float]` longitude
* `--debug=[bool: [true|false]]` show mouse in browser
* `--delay=[int]` seconds waiting to load google maps, default: 1


## example: 
`node app.js --lat=48.7451191 --long=9.3222745  --delay=10`
