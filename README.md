# photo-test

add a folder named "screenshots" !


execute with:

node app.js [options]
    --lat=[float]: latitude
    --long=[float]: longitude
    --debug=[bool: [true|false]]: show mouse in browser
    --mode=[string: [maps]]: use google maps


example: 
    node app.js --lat=48.7451191 --long=9.3222745  --debug=true --mode=maps
